搜索的工作方式（面向新手）

Styles

This starter template comes pre-loaded with Tailwind CSS, a utility CSS framework that allows you to customize and build complex designs without touching a line of CSS. There are also a few base Sass files in the /source/_assets/sass folder, set up with the expectation that you can add any custom CSS into _documentation.scss.

You can re-work the architecture of the Sass includes any way you’d like; just make sure to keep the @tailwind references in your final files.

如果您有兴趣改善您的网站在 Google 搜索中的呈现效果，并且愿意花一点时间了解搜索引擎优化 (SEO) 和 Search Console，请参阅下面的入门指南。您不需懂得 HTML 或编码，但需花一些时间思考您网站的组织和编写方式，并且愿意对您的网站做出一些更改。令人欣慰的是，您只要多付出一点点努力，就可能在很大程度上改善网站在搜索结果中的排名。 注册 Search Console。这是免费的，没有升级或付费服务；当 Google 在您的网站上发现异常活动（例如抓取问题或黑客入侵的迹象）时，您会收到提醒。 添加网站并验证您对该网站的所有权。您需要证明自己就是该网站的所有者，因为 Search Console 显示的与您网站相关的信息只应该让网站所有者知晓，而且 Search Console 允许您做出可能会影响您的网站在 Google 搜索结果中的排名的更改。 阅读我们的 Search Console 基本用法指南。初级用户通常只需每月快速检查一次网站，除非我们发现了问题并向您发送了提醒。 了解 Google 搜索运作方式的基础知识。您会了解更多关于本文提供的信息，而这会大大降低问题排查的难度。 阅读我们的 SEO 新手指南。此指南介绍了可对您的网站采用的最佳做法，确保 Google 能够找到、解读并在搜索结果中恰当地展示您的网站。其中的信息量很大，但您可以不急不忙地慢慢读完。 决定是否要聘用 SEO 专家。许多小网站的所有者都会独力完成自己的所有搜索引擎优化工作；事实上，如果您仔细阅读我们的文档，您自己就能成为 SEO 专家！但一些小网站（以及几乎所有大网站）都聘用了专业的 SEO 专家来帮助他们改善在 Google 搜索结果中的排名。如果您正在考虑聘用专业的 SEO 专家，请阅读本指南。

如果在学完基础知识后，您决定学习有关 SEO 或 Search Console 的更多高级主题，则可以按照 SEO 路线继续学习。
